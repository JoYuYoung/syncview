# 📱 SyncView 사용자 중심 동작 흐름도 (User Flow)

## 🎯 프로젝트 개요
**SyncView**는 AI 기반 다국어 뉴스 플랫폼으로, BBC 글로벌 뉴스를 실시간 요약·번역·감성 분석하여 제공합니다.

---

**🔑 인증 처리 방식:**
- ✅ 백엔드: Bcrypt 암호화 + PostgreSQL 저장
- ✅ 프론트: localStorage에 사용자 정보 저장
- ✅ 상태 관리: React Context (App.jsx)
- ✅ 세션 유지: 페이지 새로고침 시 localStorage에서 복원

```
┌──────────────────────────────────────────────────────────────────┐
│                    뉴스 피드 메인 화면                             │
│                      /newsfeed                                    │

[페이지 로드]
   │
   ▼
┌─────────────────────────┐

│                                                                  │
│  [필터 탭]                                                       │
│  • 📰 모든 뉴스 (filterMode: "all")                             │
│  • 🌟 추천 뉴스 (filterMode: "recommended")                     │
│                                                                  │
│  [뉴스 카드 x 10]                                                │

│  │ [썸네일] 제목                            │                  │
│  │          요약 (120자)                    │                  │
│  │ 😊/😟/😐 감성 아이콘                      │  ← 감성 분석     │
│  │ 🔗 중복 기사 N개                         │  ← 유사도 검출   │
│  └──────────────────────────────────────────┘                  │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘

[사용자가 기사 클릭]
   │
   ▼
┌───────────────────────────────────────────────────────────────────┐
│              우측 패널: 요약 & 번역 (60%)                         │
├───────────────────────────────────────────────────────────────────┤
│                                                                    │
│  [탭 전환]                                                         │
│  • [요약] 탭 (기본 선택)                                          │
│  • [번역] 탭                                                       │
│                                                                    │
│  ┌────────────────────────────────────────────┐                  │
│  │ STEP 1: 요약 요청                           │                  │
│  │ GET /news/summary?url={기사 링크}          │                  │
│  │                                             │                  │
│  │ → 웹 스크래핑 (BeautifulSoup)              │                  │
│  │ → AI 요약 (BART-Large-CNN)                 │                  │
│  │ → 요약문 반환 (max 130단어)                │                  │
│  └─────────────┬───────────────────────────────┘                  │
│                │                                                   │
│                ▼                                                   │
│  ┌─────────────────────────────────────────────┐                 │
│  │ 😊 감성 분석 자동 실행                      │  ← 클라이언트   │
│  │ (키워드 기반 긍정/부정/중립 판별)           │     측에서 처리  │
│  │ 신뢰도 점수 표시                             │                 │
│  └─────────────────────────────────────────────┘                 │
│                                                                    │
│  ┌─────────────────────────────────────────────┐                 │
│  │ 🔗 유사 기사 3개 표시                       │                 │
│  │ (같은 이슈를 다룬 다른 매체 기사)            │                 │
│  │ • 유사도 75-95%                             │                 │
│  │ • 클릭 시 해당 기사로 이동                   │                 │
│  └─────────────────────────────────────────────┘                 │
│                                                                    │
└───────────────────────────────────────────────────────────────────┘

[사용자가 [번역] 탭 클릭]
   │
   ▼
┌─────────────────────────────────────────────┐
│ STEP 2: 번역 요청                            │
│ POST /api/translate                         │
│ { text: "요약문", target_lang: "ko" }       │
│                                              │
│ → NLLB-200 모델 로딩 (Lazy)                │
│ → 텍스트 청킹 (700자 단위)                  │
│ → 영어 → 한국어 번역                        │
│ → 캐싱 (동일 요청 시 즉시 반환)             │
│                                              │
│ [번역 결과 표시]                             │
│ • 한국어로 번역된 요약문                     │
│ • 로딩 스피너 표시 (1-3초)                  │
└─────────────────────────────────────────────┘

[원문 보기 버튼]
   │
   ▼
새 탭에서 BBC 원문 기사 열기
```

**⚙️ 기술 스택 상세:**

| 단계 | 기술 | 설명 |
|------|------|------|
| RSS 수집 | `feedparser` | BBC RSS XML 파싱 |
| 웹 스크래핑 | `BeautifulSoup` | 기사 본문 추출 |
| 요약 | `BART-Large-CNN` | Hugging Face Transformers |
| 번역 | `NLLB-200-distilled-600M` | Meta AI 다국어 모델 |
| 감성 분석 | 키워드 매칭 | 클라이언트 측 처리 |
| 캐싱 | `@lru_cache(512)` | Python functools |

## 🎨 사용자 인터랙션 시나리오

### 시나리오 1: 신규 사용자 온보딩

```
👤 사용자: "김철수" (30대, IT 업계 종사자)
🎯 목표: 글로벌 경제 뉴스를 빠르게 파악하고 싶음

1. SyncView 접속 → Welcome 페이지
   ├─ "글로벌 뉴스를 한눈에" 메시지 확인
   └─ [지금 바로 시작] 버튼 클릭

2. 회원가입 페이지
   ├─ 이메일: kim@example.com
   ├─ 비밀번호: ********
   ├─ 관심분야: "경제" 선택  ← ✨ 개인화 시작
   └─ [가입하기] 클릭

3. 자동 로그인 → Welcome 페이지 리다이렉트
   └─ 상단에 "kim@example.com" 표시 확인

4. [구독] 메뉴 클릭 → /subscription
   ├─ BBC, Reuters 선택
   ├─ 경제, 기술 카테고리 선택
   └─ [저장] 클릭

5. [홈] 메뉴 클릭 → /newsfeed
   ├─ 상위 3개 기사에 🌟 추천 배지
   │  (이유: "관심사 맞춤 - 경제")
   └─ 10개 BBC 기사 목록 로드

6. 첫 번째 기사 클릭
   ├─ 우측에 영어 요약문 표시 (3초 소요)
   ├─ 😊 긍정 감성 자동 표시
   └─ 🔗 유사 기사 3개 하단에 표시

7. [번역] 탭 클릭
   ├─ 한국어 번역 시작 (2초 소요)
   └─ 번역된 요약문 표시

8. [북마크] 아이콘 클릭
   └─ 로컬스토리지에 저장

9. [원문 보기] 버튼 클릭
   └─ 새 탭에서 BBC 기사 원문 열람
```

## 📊 데이터 흐름도 (Data Flow)

```
┌──────────────────────────────────────────────────────────────┐
│                      백엔드 (FastAPI)                          │
├──────────────────────────────────────────────────────────────┤
│                                                               │
│  [데이터 소스]                                                │
│  ├─ BBC RSS Feed (http://feeds.bbci.co.uk/news/world/rss.xml)│
│  ├─ PostgreSQL (사용자 정보, 관심사)                          │
│  └─ 로컬 AI 모델 (BART, NLLB)                                │
│                                                               │
│  [API 엔드포인트]                                             │
│  ├─ POST /auth/register   → DB INSERT                        │
│  ├─ POST /auth/login      → DB SELECT + Bcrypt 검증          │
│  ├─ GET  /news/bbc        → RSS 파싱                         │
│  ├─ GET  /news/summary    → 웹 스크래핑 + BART 요약          │
│  └─ POST /api/translate   → NLLB 번역 + LRU 캐시             │
│                                                               │
└───────────────────────┬──────────────────────────────────────┘
                        │
                        │ HTTP/JSON
                        │
                        ▼
┌──────────────────────────────────────────────────────────────┐
│                   프론트엔드 (React + Vite)                    │
├──────────────────────────────────────────────────────────────┤
│                                                               │
│  [상태 관리]                                                  │
│  ├─ App.jsx: user 상태 (로그인 정보)                         │
│  ├─ NewsFeed.jsx: articles, summary, translation             │
│  └─ localStorage: user 정보, bookmarks                        │
```
