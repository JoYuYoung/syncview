# 프로젝트명: SyncView - AI 기반 뉴스 분석 플랫폼

## 1. 프로젝트 개요
- BBC 뉴스를 실시간으로 수집하고 AI로 분석하는 웹 애플리케이션
- 뉴스 요약, 번역, 감성 분석, 데이터 시각화 기능 제공
- 사용자 맞춤형 뉴스 큐레이션

## 2. 백엔드 기술 스택 (Python)

### 프레임워크
- FastAPI (웹 프레임워크)
- Uvicorn (ASGI 서버)
- SQLAlchemy (ORM)
- PostgreSQL (데이터베이스)

### AI/ML 라이브러리
- PyTorch (딥러닝)
- Transformers (Hugging Face)
  - facebook/bart-large-cnn (뉴스 요약)
  - Helsinki-NLP/opus-mt-en-ko (영어→한국어 번역)
- SentencePiece (토크나이저)

### 웹 스크래핑
- BeautifulSoup4 (HTML 파싱)
- Feedparser (RSS 파싱)
- Requests (HTTP 요청)

### 보안
- Passlib + bcrypt (비밀번호 암호화)
- CORS Middleware

## 3. 프론트엔드 기술 스택 (JavaScript)

### 프레임워크 & 라이브러리
- React 18.3.1
- React Router DOM 6.30.1
- Vite (빌드 도구)

### UI/스타일링
- TailwindCSS 3.4.17
- PostCSS + Autoprefixer
- 글래스모피즘 디자인
- 그라데이션 배경 + 블롭 애니메이션

### 데이터 시각화
- Recharts 3.3.0
  - PieChart (도넛 차트)
  - LineChart (라인 차트)
  - BarChart (바 차트)
  - 인터랙티브 툴팁

## 4. 주요 기능

### 사용자 관리
- 회원가입/로그인 (이메일 기반)
- 비밀번호 bcrypt 암호화
- 관심 분야 선택 (경제, 정치, 연예)
- 프로필 관리

### 뉴스 피드
- BBC RSS 실시간 크롤링
- 뉴스 목록 (제목, 요약, 썸네일, 발행시간)
- 추천 뉴스 필터링
- 중복/유사 기사 표시

### AI 분석
- **자동 요약**: BART 모델로 뉴스 요약
- **번역**: Marian 모델로 영→한 번역
- **감성 분석**: 긍정/부정/중립 자동 분류
- 유사 기사 추천

### 데이터 시각화 대시보드 (Analytics)
1. **통계 카드**: 총 읽은 기사, 긍정 비율, 인기 카테고리, 읽기 속도
2. **감성 분석 통계**: 도넛 차트로 긍정/부정/중립 비율
3. **읽기 활동 통계**: 바 차트로 요일별 읽은 기사 수
4. **시간대별 감성 트렌드**: 라인 차트로 감성 변화 추이
5. **카테고리별 분포**: 가로 바 차트로 주제별 기사 수
6. **AI 인사이트**: 데이터 기반 자동 인사이트 생성

### 기타 기능
- 북마크 저장
- 뉴스 구독 설정
- 알림 시스템

## 6. 데이터 흐름

### 뉴스 수집
BBC RSS → FastAPI (feedparser) → React (NewsFeed.jsx)

### AI 요약
사용자 클릭 → BeautifulSoup (본문 추출) → BART (요약) → React 표시

### 번역
요약 텍스트 → Marian 모델 (영→한) → React 표시

### 감성 분석 & 시각화
요약 텍스트 → 키워드 분석 → 긍정/부정/중립 → Recharts 시각화

## 7. UI/UX 디자인

### 스타일링 특징
- TailwindCSS 유틸리티 클래스
- 글래스모피즘: `backdrop-blur-lg bg-white/80`
- 그라데이션 배경: `bg-gradient-to-br from-blue-50 via-sky-100 to-indigo-100`
- 블롭 애니메이션: 부드러운 배경 움직임
- 호버 효과: `hover:scale-105 transition-all`

### 색상 시스템
- 메인: 블루(#3B82F6), 인디고(#6366F1)
- 감성 분석: 긍정(그린), 중립(그레이), 부정(레드)
- Analytics: 퍼플-핑크 그라데이션

## 8. 실행 방법

### 백엔드
```bash
cd syncview_backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
uvicorn main:app --reload
```

### 프론트엔드
```bash
cd syncview_frontend
npm install
npm run dev
```

### 접속 URL
- 프론트엔드: http://localhost:5173
- 백엔드 API: http://localhost:8000
- API 문서: http://localhost:8000/docs
